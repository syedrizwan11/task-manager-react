
# Task Management System – Frontend

This is the frontend application for the **Task Management System**, built using **React, TypeScript**, and **Tailwind CSS**.  
It provides role-based dashboards for users and admins with secure authentication.

---

## 🚀 Tech Stack

- React
- TypeScript
- React Router v6
- Tailwind CSS
- Axios
- Context API (Global Auth State)

---

## 📁 Features

- Login / Logout with HTTP-only cookie auth
- Role-based routing (Admin / User)
- Protected routes using layouts
- Admin dashboard with statistics
- Task creation, assignment, and completion
- Profile update with live auth revalidation
- Clean, feature-based folder structure
